How to Run the Space Shooter Game

Follow these steps to run the game on your system:

1. Install Pygame

Open your terminal or command prompt and run:

pip install pygame


This will install the Pygame library required for the game.

2. Download the Project

Clone the GitHub repository or download it as a ZIP file:

git clone https://github.com/TCLingesh/Space_Shooter_Game.git

If downloaded as ZIP, extract the folder.

3. Check the Assets Folder

Make sure all game images are present:

Enemy ships

Player ship

Lasers

Background image

These must match the file names in the code.

4. Run the Project.py file

5. Play the Game

Move using WASD or Arrow Keys

Shoot using SPACE

Avoid enemy lasers

Destroy enemy ships

Try to survive as long as possible
